# pos2ecf

a tool which could transfer POS to ECEF


## Dependencies

* Eigen

##  Compilation
prerequisites: cmake version>=3.0

```
1. git clone https://github.com/AndrewAndJenny/pos2ecf.git
2. cd pos2ecf
```

**in the cmake-bash**
* Windows
```
1. mkdir build && cd build
2. cmake .. -A x64
3. open the vs solution, and install
```
* Linux
```
1. mkdir build && cd build
2. cmake ..
3. make
```
